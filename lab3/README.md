# Hangman — Lab 3 (Settings + localStorage + Form + Portal)

## Реалізовано вимоги
- Налаштування гри: складність, кількість спроб (елементів), таймер (швидкість tick), фільтр довжини слова
- Зберігання налаштувань через **localStorage** + **Context**
- Обов'язкова форма налаштувань: **react-hook-form + yup** (валідація)
- **Portal**: діалогове вікно завершення гри (рендериться в `#modal-root`)
  - показує інформацію про завершення
  - дозволяє: **наступний тур** або **цей тур заново**, також "Деталі" та "На старт"

## Запуск
```bash
npm install
npm run dev
```
