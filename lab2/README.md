# Hangman — Lab 2 (React + Hooks)

## Вимоги, які реалізовано
- 3 сторінки без роутингу: Start / Game / Result (перемикаються станом)
- Основна логіка гри реалізована через **кастомні хуки**, щоб компоненти були чистими
- Гравець відгадує слово по літерах. Невірні літери зменшують спроби
- Виграш/програш визначається автоматично
- Підтримка введення: кліки по екранній клавіатурі + фізична клавіатура A–Z

## Запуск
```bash
npm install
npm run dev
```

## Архітектура
- `src/hooks/` — вся логіка (useHangmanApp, useHangmanGame, useKeyPress, useStartSettings)
- `src/lib/` — чисті утиліти (maskWord, pickRandomWord, attempts)
- `src/pages/` — сторінки (майже без логіки)
- `src/widgets/` — більші UI блоки
- `src/ui/` — перевикористовувані компоненти
